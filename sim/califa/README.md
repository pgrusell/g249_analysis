# CALIFA p2p simulation

In this simulation we will generate, using INCL, the following reaction $^{25}F(p,2p)^{24}O$ in the realistic CAD model of the vacuum chamber, surrounded by CALIFA.

For the digitization of CALIFA we set the values crystal by  crystal using the file `califaDigi.par`. Since we are only here to see the effect of the protons, the $\gamma$-response part will be set by default. However, realistic ToT efficiencies are considered there.
